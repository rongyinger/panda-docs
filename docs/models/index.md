# 模型总览

熊猫算力持续同步上游官方最新模型，价格与官方保持同步或更低。以下为当前可用模型列表（更新于 2026-05）。

## OpenAI 系列

| 模型 ID | 上下文 | 能力特点 | 适用场景 |
|---------|--------|---------|---------|
| `gpt-4o` | 128K | 多模态旗舰 | 复杂推理、图片理解 |
| `gpt-4o-mini` | 128K | 轻量快速 | 日常对话、简单任务 |
| `o1` | 200K | 深度思维链 | 数学、代码、逻辑推理 |
| `o3-mini` | 200K | 快速推理 | 编程、科学计算 |

## Anthropic Claude 系列

| 模型 ID | 上下文 | 能力特点 | 适用场景 |
|---------|--------|---------|---------|
| `claude-opus-4-7` | 200K | 最强综合能力 | 复杂分析、长文处理 |
| `claude-sonnet-4-6` | 200K | 性能均衡 | 多数生产场景 |
| `claude-haiku-4-5` | 200K | 极速低成本 | 高频简单任务 |

## Google Gemini 系列

| 模型 ID | 上下文 | 能力特点 | 适用场景 |
|---------|--------|---------|---------|
| `gemini-2.5-pro` | 1M | 超长上下文旗舰 | 超长文档处理 |
| `gemini-2.5-flash` | 1M | 速度优先 | 实时对话、摘要 |

## 选型建议

| 需求场景 | 推荐模型 |
|---------|---------|
| 通用对话，追求低成本 | `gpt-4o-mini` 或 `claude-haiku-4-5` |
| 代码生成与调试 | `claude-sonnet-4-6` 或 `o3-mini` |
| 复杂推理 / 数学 | `o1` 或 `claude-opus-4-7` |
| 超长文档（>100K token） | `gemini-2.5-pro` |
| 图片理解 | `gpt-4o` 或 `claude-opus-4-7` |

::: tip 如何选择
成本敏感场景首选 Flash/Mini 系列；追求效果首选 Opus/Pro 系列；代码和推理任务 o1/o3 系列表现突出。
:::

更详细的能力对比请阅读 [如何选择合适的 AI 模型](/blog/how-to-choose-model)。
