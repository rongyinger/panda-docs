# 拓展连接

熊猫算力平台支持与主流 AI 创作工具无缝集成，让你的工作流更高效。

## 支持连接的工具

| 工具 | 类型 | 教程 |
|------|------|------|
| [ComfyUI](/connect/comfyui) | 图像生成 | ✅ 已有教程 |
| Stable Diffusion WebUI | 图像生成 | 见使用教程 |
| Open WebUI | LLM 对话 | 即将上线 |
| n8n | 工作流自动化 | 即将上线 |

## 通用连接方式

大多数工具都支持通过修改 API 地址的方式接入熊猫算力：

```
Base URL: https://api.pandatoken.com/v1
API Key:  你的 PandaToken API Key
```
