## ✨ 新功能

### 企业版：独立 Key 池隔离（2026-04-25）

企业版账号现支持在主账号下创建多个「子项目」，实现：

- API Key 完全隔离，子项目间无法互用
- 独立的 RPM/TPM 限速配置
- 分项目的用量报表与账单
- 预算告警（触达阈值后自动发邮件通知）

配置入口：控制台 → 项目管理 → 新建子项目

## 🔧 问题修复

### 修复 Streaming 高并发下偶发断流（2026-04-18）

**现象**：在高并发场景下（>100 并发连接），少量 SSE 连接会在中途被强制断开，客户端收到不完整响应。

**原因**：网关层的 keep-alive 超时设置过短，与上游连接池的配置不匹配。

**修复**：调整网关超时配置，并增加了连接健康检测机制。修复后内部压测未复现该问题。
