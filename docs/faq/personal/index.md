# 个人用户常见问题

本页面收录个人开发者在使用熊猫算力过程中最常遇到的问题。

## 账号与充值

**Q：注册后有免费额度吗？**

有。新用户注册后自动获得 ¥5 免费额度，无需充值即可体验主流模型。

**Q：最低充值金额是多少？**

最低充值 ¥10，无月费、无有效期限制（账号连续 12 个月无活跃记录除外）。

**Q：支持哪些付款方式？**

支持微信支付、支付宝。如需发票请在充值时勾选开票选项。

---

## 接入与使用

**Q：我用的是 OpenAI 官方 SDK，能直接换成熊猫算力吗？**

完全兼容。只需修改两处：

```python
# 原来
client = OpenAI(api_key="sk-...")

# 修改为
client = OpenAI(
    api_key="你的 PandaToken Key",
    base_url="https://api.pandatoken.com/v1"
)
```

**Q：支持哪些模型？**

请参考 [模型总览](/models/) 页面，持续更新。

**Q：有请求频率限制吗？**

默认限速：60 RPM（每分钟请求数），如需提升请在控制台申请或联系客服。

---

## 计费说明

**Q：如何计算费用？**

按 Token 计费，输入和输出分别计价，与官方价格保持同步或更低。详见控制台「价格计算器」。

**Q：余额不足会怎样？**

请求会返回 `402 Insufficient Balance` 错误，不会产生欠费。
