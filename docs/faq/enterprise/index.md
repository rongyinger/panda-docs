## 商务合作

**Q：如何申请企业版？**

请发送邮件至 `enterprise@pandatoken.com`，或通过右下角悬浮按钮扫码联系商务，我们会在 1 个工作日内回复。

**Q：企业版最低消费是多少？**

企业版采用预付费充值制，最低充值 ¥500，余额长期有效。按实际用量消耗，无固定月费。

**Q：支持签合同和开专票吗？**

支持。可签署正式服务合同，提供增值税专用发票（税点 6%），支持对公银行转账。

## 技术方案

**Q：支持私有化部署吗？**

支持。我们提供 Docker 镜像 + Kubernetes Helm Chart 的部署方案，可部署到客户自有机房或云环境。私有化部署方案需单独签署协议，请联系商务。

**Q：如何实现多项目、多团队的 Key 隔离？**

企业版支持在一个主账号下创建多个「子项目」，每个子项目独立分配：
- API Key
- 用量限额
- 费用预算告警

各子项目的用量和账单完全独立，方便按部门或项目核算成本。

**Q：如何保障数据安全？**

- 请求数据不落盘，过境即销毁
- 支持签署数据处理协议（DPA）
- 企业专线方案可实现流量不经公共互联网
