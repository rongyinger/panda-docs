# 常见问题

## 热门问题

- [显存不足导致任务失败怎么办？](/faq/vram-issue)
- API Key 泄露了怎么办？→ 立即前往控制台「API Key 管理」吊销并重建
- 支持哪些付款方式？→ 微信支付、支付宝；企业客户支持对公转账
- 如何查看用量账单？→ 控制台 → 用量统计 → 按日 / 月查看

## 账号与充值

**Q：注册后有免费额度吗？**

新用户注册后自动获得 ¥5 免费额度，可用于体验主流功能。

**Q：最低充值金额是多少？**

最低充值 ¥10，无月费，余额长期有效。

**Q：如何申请发票？**

充值时勾选「需要发票」，填写抬头信息，系统会在到账后自动开具电子发票。

## 实例与算力

**Q：实例关机后还会收费吗？**

关机状态下仅收取磁盘存储费用，不收取 GPU 费用。建议不使用时关机，长期不用时释放实例。

**Q：如何在实例间共享数据？**

推荐使用「共享存储」（类似 NAS），将数据盘挂载到多个实例，避免重复下载模型文件。
