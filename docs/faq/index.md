## 热门问题

1. [为什么响应速度比官方 API 慢？](/faq/personal/why-slower-than-official)
2. API Key 泄露了怎么办？ → 立即前往控制台「API Key 管理」吊销并重新生成
3. 支持哪些付款方式？ → 支持微信、支付宝、银行卡；企业客户支持对公转账
4. 如何查看用量账单？ → 登录控制台 → 用量统计 → 按日/月查看明细
